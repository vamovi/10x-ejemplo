# basic-node-server

Servidor ultra básico con Node que sirve archivos estáticos desde `public/`.

## Cómo correr

```bash
cd basic-node-server
npm start
```

Luego abre `http://localhost:3000`.

## Estructura

- `server.js`: servidor HTTP (sin Express)
- `public/index.html`: página principal
- `public/app.js`: JS del navegador
